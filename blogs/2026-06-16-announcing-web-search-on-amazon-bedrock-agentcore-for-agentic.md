Amazon Bedrock AgentCore launches Web Search in general availability, a fully managed tool enabling agents to ground responses in current web knowledge while maintaining data residency within secured AWS environments. Built on Amazon's search infrastructure behind Alexa+, Amazon Q Business, and Kiro, results include rankings, snippets, source URLs, titles, and publication dates.
