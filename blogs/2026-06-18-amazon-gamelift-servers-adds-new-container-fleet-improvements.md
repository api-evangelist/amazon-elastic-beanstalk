Amazon GameLift Servers enhances container fleets with Linux capability customization and inter-container communication improvements. Developers can customize Linux capabilities like NET_RAW for custom networking or SYS_PTRACE for debugging, and the new ListContainersNetworkInfo() server SDK action retrieves container identity, local IP addresses, and group types for automatic service discovery.
