AWS DevOps Agent introduces release management in preview, evaluating code changes for production readiness and executing autonomous release testing across AWS, multicloud, and on-premises environments. Release readiness review examines changes against internal standards, dependency effects, and access controls, while release testing generates and executes test plans for web and API applications.
