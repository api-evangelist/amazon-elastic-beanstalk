AWS announces general availability of the managed agent harness in Amazon Bedrock AgentCore, providing orchestration, tool execution, context window management, cross-session state persistence, failure recovery, and session isolation. Configuration-based definition of model, tools, skills, and instructions enables production-grade agents in minutes, with Strands export available and Claude Agent SDK export coming soon.
