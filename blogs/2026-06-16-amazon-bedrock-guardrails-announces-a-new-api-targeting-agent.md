Amazon Bedrock Guardrails introduces the InvokeGuardrailChecks API, a resourceless interface enabling individual safeguard application at any agentic workflow point without creating a guardrail resource. The API provides per-request control with numeric severity and confidence scores, operating in detect-only mode and supporting content filters, prompt attack detection, and sensitive information filtering.
