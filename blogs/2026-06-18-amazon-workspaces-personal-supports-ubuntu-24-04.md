AWS announces an Ubuntu 24.04 LTS bundle for Amazon WorkSpaces Personal in the AWS China (Ningxia) Region operated by NWCD. The bundle delivers updated Linux packages, toolchains, security enhancements, and an extended support lifecycle, providing a migration pathway for customers running older Amazon Linux 2 WorkSpaces.
