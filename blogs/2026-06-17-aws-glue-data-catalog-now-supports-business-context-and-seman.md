AWS Glue Data Catalog enables business context and semantic search in preview, helping users discover data by semantic meaning through glossary terms and custom metadata fields via the new Glue Search API. MCP-compatible agents including Claude Code, Kiro, and Cursor can integrate using the aws-data-analytics plugin from the Agent Toolkit for AWS.
