AWS Security Agent (now part of AWS Continuum) adds Kiro and Claude Code support to trigger security scans from development environments. Simulated validations execute scanner findings in isolated sandboxes to provide exploit proof and reduce false positives, with new integrations for GitLab.com, GitLab Self Managed, GitHub Enterprise, Bitbucket, and Confluence.
