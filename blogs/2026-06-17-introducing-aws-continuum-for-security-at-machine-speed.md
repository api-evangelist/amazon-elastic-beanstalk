AWS introduces Continuum, discovering, prioritizing, validating, and remediating security risks at machine speed within user-defined guardrails. It ingests findings from existing tools, prioritizes using context graphs, and validates exploitability in isolated sandboxes; AWS Security Agent penetration testing and code scanning now function as Continuum offerings, with STRIDE-formatted threat modeling in preview.
