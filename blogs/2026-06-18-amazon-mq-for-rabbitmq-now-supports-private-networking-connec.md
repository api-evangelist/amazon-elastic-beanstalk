Amazon MQ for RabbitMQ introduces private networking, enabling broker connections to private VPC resources without public exposure for Federation, Shovel, and authentication scenarios with LDAP and OAuth 2.0 providers. The implementation leverages Amazon VPC Lattice, AWS Resource Access Manager, and AWS PrivateLink, available in all Regions supporting VPC Lattice.
