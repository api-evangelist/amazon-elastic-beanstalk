AWS Glue Interactive Sessions add Apache Spark Connect support, enabling Spark application development from Amazon SageMaker Unified Studio managed notebooks, Jupyter, or Visual Studio Code while using Glue serverless infrastructure without cluster management. Monitoring occurs through Spark UI, with history via Spark History Server and session management through AWS APIs.
