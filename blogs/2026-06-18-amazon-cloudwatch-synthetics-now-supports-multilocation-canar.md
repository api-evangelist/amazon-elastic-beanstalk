Amazon CloudWatch Synthetics introduces multilocation canaries, letting developers run identical monitoring tests across multiple AWS Regions simultaneously from a single management point. A canary created in one primary Region automatically replicates to chosen additional Regions, consolidating metrics and artifacts centrally, and existing single-region canaries can be upgraded without recreation.
