title: "Amazon RDS for SQL Server increases the maximum size and provisioned performance of General Purpose (gp3) volumes"
url: "https://aws.amazon.com/about-aws/whats-new/2026/06/rds-sqlserver-increases-gp3-limits/"
date: "2026-06-18"
feed_url: "https://aws.amazon.com/about-aws/whats-new/recent/feed/"
---
Amazon RDS for SQL Server now supports enhanced gp3 volume limits, scaling each volume to 64 TiB (4X), 80,000 IOPS (5X), and 2,000 MiB/s throughput (2X). Customers can configure up to three gp3 or io2 volumes per instance, increasing total capacity to 256 TiB per instance with no pricing changes.
