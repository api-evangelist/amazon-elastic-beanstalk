AWS Security Agent (now part of AWS Continuum) adds threat modeling, an AI-powered agentic capability that automatically generates threat models from design documents or source code. Available in public preview, it identifies threats and recommends STRIDE framework mitigations, with integration into IDEs like Kiro and Claude Code at no additional cost during preview.
