date: "2026-06-17"

AWS introduces bmn-cx3a instances on second-generation AWS Outposts racks, featuring 5th Generation AMD EPYC processors up to 4.1 GHz with NVIDIA ConnectX-7 NICs delivering up to 800 Gbps of bare-metal accelerated network bandwidth. Instances offer up to 256 cores and 1.5 TB memory across two sizes, with native Layer 2 multicast and hardware Precision Time Protocol support.
