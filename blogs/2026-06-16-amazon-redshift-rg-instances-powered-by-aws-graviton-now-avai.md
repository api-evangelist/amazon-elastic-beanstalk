Amazon Redshift expands Graviton-based RG instance availability to Africa (Cape Town), Asia Pacific (Bangkok), and Mexico (Central). RG instances deliver up to 4.2X better price-performance for data warehouse workloads, run up to 2.4x faster than previous RA3 instances, and cost 30% less per vCPU, offering rg.xlarge and rg.4xlarge node types.
