Amazon SageMaker AI introduces observability for inference endpoints, tracking token performance, GPU health, inference component placement, and autoscaling behavior. Metrics like Time to First Token, inter-token latency, queue depth, and tokens per second surface in a pre-built SageMaker AI Insights dashboard in CloudWatch, with Grafana integration via regional PromQL endpoints.
