Oracle Database@AWS extends support to Oracle Autonomous AI Database Serverless (ADB-S), a fully managed Oracle service on Exadata that provisions directly from the AWS Management Console, CLI, or APIs. Four workload types scale compute and storage independently, with Autonomous Data Guard, automated S3 backups, cross-Region disaster recovery, AWS KMS encryption, and CloudWatch monitoring.
