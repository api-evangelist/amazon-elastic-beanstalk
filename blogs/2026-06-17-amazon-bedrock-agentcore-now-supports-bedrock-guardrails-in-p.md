Amazon Bedrock AgentCore now integrates Bedrock Guardrails in policy, combining AgentCore's authorization capability with Guardrails evaluating every authorized action output and gateway target call input in real time. Guardrails detect and block prompt injection attacks, harmful content, and sensitive information exposure, with policy evaluation at the gateway perimeter outside agent code.
